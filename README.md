# DigiTools –A Premium Digital Tools Marketplace

## Project Overview

DigiTools is a modern single-page web application built with React that showcases a collection of premium digital tools. Users can browse products, add them to a cart, remove items, and proceed to checkout — all with a smooth and interactive user experience.

---

## Features

### Product Browsing

- Display of multiple digital products in a responsive 3-column layout
- Each product includes name, description, price, features, and tags

### Cart Functionality

- Add products to cart
- Remove products from cart
- Dynamic cart count in Navbar
- Total price calculation
- “Proceed to Checkout” clears the cart

### Responsive Design

- Fully responsive across mobile, tablet, and desktop

---

## Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- React-Toastify
